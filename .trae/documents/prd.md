# Raq.Store 兑换店预告页面 - 产品需求文档

## 1. 项目概述

### 1.1 产品定位
Raq.Store 兑换店的预售预告静态网页，用于在产品正式上线前吸引用户关注，传达品牌信息，并建立信任感。

### 1.2 目标用户
- Raqcoin 持有者及潜在兑换用户
- BNB Chain 生态系统参与者
- 加密货币交易用户

### 1.3 核心价值
- 预告兑换店即将上线的信息
- 展示项目可信度（合约地址、技术栈）
- 国际化用户体验

## 3. 设计规范

### 3.1 设计风格
- **整体风格**: 金融级黑色主题，类似 OKX 交易所的简洁专业风格
- **色彩体系**:
  - 背景色: 深黑 (#0A0A0A) 至 深灰 (#1A1A1A)
  - 主色调: 白色/浅灰文字
  - 强调色: 金色/琥珀色 (#D4AF37) 用于关键信息和交互元素
  - 辅助色: 冷灰 (#888888) 用于次要文字
- **字体**: 使用现代无衬线字体，确保多语言兼容性
- **视觉元素**: 微妙的渐变、光效和几何装饰元素

### 3.3 交互设计
- 语言切换: 右上角下拉菜单，点击切换全站语言
- 合约地址: 可点击复制
- 页面加载动画: 优雅的渐入效果
- 背景装饰: 微妙的粒子或线条动画

## 4. 技术要求

### 4.1 技术栈
- HTML5 + CSS3 + 原生 JavaScript
- 无需框架，保持轻量
- 响应式设计，支持移动端和桌面端

### 4.2 部署方案
- GitHub Pages 部署
- 支持自定义域名配置

### 4.3 性能要求
- 首屏加载 < 2秒
- 完全静态页面
- 无外部依赖（除字体外）
